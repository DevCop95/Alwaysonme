# 🕐 Alwaysonme · Reloj Digital Cartagena

Reloj digital en tiempo real con diseño minimalista fullscreen. Fuente DSEG (estética LED/LCD retro), optimizado para móvil.

## ✨ Características

- Hora en tiempo real (zona Cartagena, Colombia)
- Fuentes DSEG7 y DSEG14 — estética LCD
- Fondo negro total, fullscreen en móvil
- PWA-ready (`apple-mobile-web-app-capable`)
- Sin dependencias · 192 líneas de código
